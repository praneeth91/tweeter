# Tweeter Project

Tweeter is a single page Ajax based twitter clone using HTML5 , CSS.

## Dependencies

- Node.js
- Express
- body-parser
- chance

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
